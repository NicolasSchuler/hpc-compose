# hpc-compose Agent Manifests

This directory contains optional agent-runtime interface metadata for the
`hpc-compose` skill bundle.

- `openai.yaml` describes the OpenAI/Codex-facing surface: display name, short
  description, and the default prompt offered when a user wants help setting up
  hpc-compose on Slurm.

The workflow source of truth remains `../SKILL.md` and the progressively loaded
references under `../references/`.
