# hpc-compose Brand Assets

These files provide the tracked brand kit for README, docs, release notes, and conference or paper material.

## Assets

| File | Use |
| --- | --- |
| `hpc-compose-logo.png` | Full logo and wordmark for README-style hero placement. |
| `hpc-compose-mark.png` | Square mark for icons, favicons, and compact layouts. |
| `hpc-compose-wordmark-on-light.png` | Wordmark for light backgrounds. |
| `hpc-compose-wordmark-on-dark.png` | Wordmark for dark backgrounds. |
| `hpc-compose-social-preview.png` | `1200x630` social preview for GitHub and link sharing. |

## Palette

| Role | Hex |
| --- | --- |
| Teal service layer | `#2dd4bf` |
| Deep teal | `#19a7b3` |
| Slate layer | `#344a66` |
| Navy layer | `#263a57` |
| Text navy | `#263244` |
| Dark background | `#07111f` |

## Usage

- Prefer the full logo for README and landing-page hero placement.
- Prefer the square mark for favicons, sidebar icons, and compact references.
- Use the social preview for the GitHub repository social image. GitHub stores that setting outside the repository, so upload `hpc-compose-social-preview.png` manually in repository settings when refreshing the public preview.
